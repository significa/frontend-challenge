# :computer: How to run

```bash
# Clone Repository
$ git clone -b vinicios-silva https://github.com/VinewZ/Significa-front-challenge.git

# Open directory with the code
$ cd my-code

# Install Dependencies and Run
$ yarn wrap
```
Go to http://localhost:3000/ to see the result.
