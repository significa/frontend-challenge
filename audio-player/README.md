# Audio player

In this challenge you will be building an audio player.

## Main skills

1. UI perfectionism
2. Attention to detail

## What is expected

An audio player able to

- Play an pause audio files
- Next and previous
- Volume control (optional)
- Shuffle (optional)
- Loop (optional)
